# PRODUCERS AVENUE - UI/UX DESIGN

## Design Philosophy

Producers Avenue aims for a **clean, modern, and professional** aesthetic that appeals to music industry professionals. The design should be:
- **Intuitive:** Users find what they need without confusion
- **Fast:** Minimal loading times, snappy interactions
- **Accessible:** WCAG 2.1 AA compliant
- **Consistent:** Reusable components, predictable patterns
- **Responsive:** Works beautifully on mobile, tablet, and desktop

## Homepage Layout

### Hero Section

**Desktop:**
- Full viewport height (100vh) or fixed height (600px)
- Background: Animated music-related visual (waveform, vinyl spinning, etc.)

**Random Video Selection:** Pick one video randomly from multiple options on each page load

**Random Video Implementation:**
```javascript
// Array of hero background videos
const heroVideos = [
  '/videos/hero-1.mp4',
  '/videos/hero-2.mp4',
  '/videos/hero-3.mp4',
  '/videos/hero-4.mp4',
  '/videos/hero-5.mp4',
  '/videos/hero-6.mp4',
  '/videos/hero-7.mp4',
  '/videos/hero-8.mp4'
];

// Select random video on component mount
const randomVideo = heroVideos[Math.floor(Math.random() * heroVideos.length)];
```

**Video Specifications:**
- Resolution: 1920x1080 (or 1280x720 for smaller file size)
- Duration: 15-30 seconds max
- File size: Target 2-3MB per video (compressed)
- Format: MP4 (H.264 codec)
- Autoplay: Yes (muted)
- Loop: Yes
- Controls: No

**Performance Notes:**
- Only ONE video loads per page visit (the randomly selected one)
- Other videos in the array are never downloaded
- No performance difference between having 1 video vs. 8 videos
- Random selection happens before video loading starts

- Animation fades to 50% opacity
- Text content fades in after animation
- Content centered vertically and horizontally

**Content:**
- Main heading (text-4xl or text-5xl, bold)
  - Example: "Where Music Creators Connect, Collaborate & Thrive"
- Subheading (text-lg or text-xl, lighter weight)
  - Example: "Join the ultimate platform for producers, musicians, and music professionals"
- Search bar (large, prominent, 500px width)
  - Placeholder: "Find members, products, or services..."
- CTA buttons:
  - Primary: "Get Started" (large, primary color)
  - Secondary: "Learn More" (outline style)

**Mobile:**
