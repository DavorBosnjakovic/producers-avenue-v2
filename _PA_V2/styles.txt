## ğŸ¨ COLOR SYSTEM

### Primary Brand Color
```
Primary Blue: #009ae9
Hover: #0088d1
Active: #0076b9
```

### Usage:
- **Primary buttons** (CTA and regular actions)
- **Links** (hover state)
- **Active states** (tabs, navigation)
- **Icons** (checkmarks, highlights)
- **Rotating word** in hero heading

### Semantic Colors
```
Success: #10b981
Warning: #f59e0b
Error: #ef4444
Info: #3b82f6
```

### Theme Colors (Light/Dark Mode) - UPDATED v1.1

**IMPORTANT: Page Backgrounds vs Card Backgrounds are DIFFERENT!**

```css
/* Light Mode */
Page Background: #fafafa (very light gray)
Card Background: rgba(255, 255, 255, 0.6) (60% white with backdrop blur)
Text Primary: #1a1a1a
Text Secondary: #666666
Border: #e0e0e0

/* Dark Mode */
Page Background: #0a0a0a (very dark - almost black)
Header/Body Background: #1a1a1a (darker gray - for header, feed cards)
Card Background: rgba(26, 26, 26, 0.6) (60% dark gray with backdrop blur)
Text Primary: #f5f5f5
Text Secondary: #b3b3b3
Border: #2a2a2a
```

### When to Use Which Background:

**Page/Main Container:**
```tsx
// For main page backgrounds (marketplace, cart, wishlist, etc.)
style={{ backgroundColor: theme === 'dark' ? '#0a0a0a' : '#fafafa' }}
```

**Regular Cards (Products, Cart Items, Order Summary):**
```tsx
// Semi-transparent with blur for glassy effect
style={{ 
  backgroundColor: theme === 'dark' 
    ? 'rgba(26, 26, 26, 0.6)' 
    : 'rgba(255, 255, 255, 0.6)',
  backdropFilter: 'blur(12px)',
  borderColor: theme === 'dark' ? '#2a2a2a' : '#e0e0e0',
}}
```

**Feed Cards (Special - with blue border and glow):**
```tsx
// These match header/body color #1a1a1a
<div className="bg-white dark:bg-[#1a1a1a] rounded-xl shadow-[0_0_8px_0_rgba(0,154,233,0.5)] border border-[#009ae9]">
```

**Solid Cards (No transparency):**
```tsx
// For sidebars, filters, etc.
style={{ backgroundColor: theme === 'dark' ? '#0a0a0a' : '#ffffff' }}
```

### How to Use Theme Colors:
```tsx
import { useTheme } from '@/lib/contexts/ThemeContext'

const { theme } = useTheme()

// For page backgrounds
style={{
  backgroundColor: theme === 'dark' ? '#0a0a0a' : '#fafafa',
}}

// For regular cards (with glassy effect)
style={{
  backgroundColor: theme === 'dark' 
    ? 'rgba(26, 26, 26, 0.6)' 
    : 'rgba(255, 255, 255, 0.6)',
  backdropFilter: 'blur(12px)',
}}

// For text
style={{
  color: theme === 'dark' ? '#f5f5f5' : '#1a1a1a',
}}
```

## ğŸ“ TYPOGRAPHY

### Font Families

**Heading Font (Bebas Neue):**
```css
font-family: var(--font-heading)
```
- All headings (h1-h6)
- "LEARN MORE" button text (uppercase)
- Section titles
- Hero animated word

**Body Font (Montserrat):**
```css
font-family: var(--font-body)
```
- All body text
- Paragraphs
- Button text (Get Started, Join Community)
- Form inputs
- Navigation items

**Usage:**
```tsx
// Heading
<h1 style={{ fontFamily: 'var(--font-heading)' }}>
  Where Music Creators
</h1>

// Body text
<p style={{ fontFamily: 'var(--font-body)' }}>
  Join the ultimate platform...
</p>

// OR use Tailwind classes
<h1 className="font-heading">Heading</h1>
<p className="font-sans">Body text</p>
```

### Font Sizes
```
Hero Heading (Desktop): 5rem (80px)
Hero Animated Word: 6rem (96px)
Hero Heading (Mobile): 3.75rem (60px)

Section Headings: text-4xl md:text-5xl lg:text-6xl
Subheadings: text-2xl md:text-3xl
Body Large: text-lg md:text-xl lg:text-2xl
Body Regular: text-base (16px)
Button Text: text-base sm:text-lg
```

## ğŸ“˜ BUTTONS

### Button Types

#### 1. CTA Button (with shine animation)
**Use for:** Primary conversion actions (Get Started, Join Community, Sign Up, Checkout)

```tsx
<Link href="/register" className="btn btn-cta">
  Get Started
</Link>
```

**Features:**
- Primary blue background (#009ae9)
- White text
- Shine animation every 5 seconds
- Rounded corners (rounded-lg)
- Padding: px-6 py-3 (default) or custom

**When to use:**
- âœ… Hero "Get Started"
- âœ… About section "Join the Community"
- âœ… Pricing page "Choose Plan"
- âœ… Cart "Proceed to Checkout"
- âœ… Any major conversion button

---

#### 2. Primary Button (no animation)
**Use for:** Regular actions (Submit, Save, Create, Buy Now)

```tsx
<button className="btn btn-primary">
  Buy Now
</button>
```

**Features:**
- Primary blue background (#009ae9)
- White text
- NO shine animation
- Rounded corners
- Same size as CTA button

**When to use:**
- âœ… Form submissions
- âœ… "Add to Cart" buttons
- âœ… "Create Product" actions
- âœ… Modal confirm buttons
- âœ… Feed "Post" button
- âŒ NOT for hero or landing page CTAs (use btn-cta)

---

#### 3. Secondary Button
**Use for:** Secondary actions (Cancel, Go Back, Learn More)

```tsx
<button className="btn btn-secondary">
  Cancel
</button>

{/* OR custom styled (like Hero "Learn More") */}
<Link 
  href="/about" 
  className="btn"
  style={{
    backgroundColor: theme === 'dark' ? '#2a2a2a' : '#f5f5f5',
    color: theme === 'dark' ? '#f5f5f5' : '#1a1a1a'
  }}
>
  Learn More
</Link>
```

**Features:**
- Gray background (light/dark mode)
- Gray text
- No animation
- Less prominent than primary

**When to use:**
- âœ… "Learn More" (Hero)
- âœ… "Continue Shopping" (Cart)
- âœ… "Cancel" actions
- âœ… "Go Back" navigation
- âœ… Secondary form actions

---

#### 4. Outline Button
**Use for:** Tertiary actions, filters, tags

```tsx
<button className="btn btn-outline">
  Filter
</button>
```

**Features:**
- Transparent background
- Primary blue border
- Primary blue text
- Hover: fills with blue

---

#### 5. Ghost Button
**Use for:** Subtle actions, menu items

```tsx
<button className="btn btn-ghost">
  Menu Item
</button>
```

### Typewriter Effect
```tsx
function TypewriterText({ text, startDelay }: { text: string; startDelay: number }) {
  const [displayedText, setDisplayedText] = useState('')
  const [hasStarted, setHasStarted] = useState(false)

  useEffect(() => {
    const startTimer = setTimeout(() => {
      setHasStarted(true)
    }, startDelay)
    return () => clearTimeout(startTimer)
  }, [startDelay])

  useEffect(() => {
    if (!hasStarted) return
    if (displayedText.length < text.length) {
      const typingTimer = setTimeout(() => {
        setDisplayedText(text.slice(0, displayedText.length + 1))
      }, 24)
      return () => clearTimeout(typingTimer)
    }
  }, [hasStarted, displayedText, text])

  return <span>{displayedText}</span>
}

// Usage
<TypewriterText 
  text="Join the ultimate platform for producers, musicians, and music professionals"
  startDelay={2000}
/>
```
