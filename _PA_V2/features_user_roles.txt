## COMMUNITY FEATURES

### User Profiles

**Profile Components:**
- Profile picture (avatar)
- Banner/header image
- Display name and username
- Bio/description (rich text, 500 characters)
- Location (city, country)
- Categories and skills (from selection)
- Genre preferences
- Experience level
- **Smart Link (rolink.me only)** - single link field
- Member since date
- Subscription badge
- Verification badges (if applicable)
- Statistics:
  - Total products/services
  - Total sales
  - Average rating
  - Followers count
  - Following count
  - Groups joined

**Smart Link Integration (Release Organizer Partnership):**
- **Single link field** - rolink.me URLs only
- **No individual social media links** - all platforms go in rolink.me Smart Link
- **Why:** Strategic partnership with Release Organizer
  - Users create Smart Link at releaseorganizer.com
  - Smart Link contains all their platforms (Instagram, SoundCloud, Spotify, etc.)
  - Clean, professional profile design
  - Cross-platform promotion
  - One link to maintain instead of many

**Profile Form:**
```
Smart Link (Release Organizer)
┌──────────────────────────────────────────────┐
│ https://rolink.me/yourlink                 │
└──────────────────────────────────────────────┘

ℹ️ Showcase all your platforms with your rolink.me Smart Link
   Create your free Smart Link at releaseorganizer.com
```

**Link Validation:**
- Only rolink.me/* URLs accepted
- Auto-prepends https:// if missing
- Error for other domains: "Only rolink.me Smart Links allowed"
- Helpful message: "Add all your platforms to your rolink.me Smart Link on https://releaseorganizer.com/ instead!"

**Profile Display:**
- Prominent button: **[View All My Links]**
- Opens rolink.me/{username} in new tab
- Clean, single-button design
- No cluttered social media icons

**Benefits:**
- Users maintain one Smart Link (easier than updating multiple fields)
- Professional appearance
- Link analytics from Release Organizer
- Cross-promotion between platforms
- Users stay in ecosystem

**Profile Privacy:**
- Public (default) - visible to all
- Members only - visible to logged-in users
- Private - visible only to connections (future feature)

**Profile Actions:**
- Follow/Unfollow
- Send message
- Send collab request
- Report profile
- Share profile (social media, copy link)

### Follow System (Hybrid Model)

**Following Mechanism:**
- **One-way following** (like Twitter/Instagram)
- Follow any public profile
- Private profiles require approval (future)
- Unfollow anytime
- Followers/Following lists viewable

**Display Logic:**
- **One-way relationship:** Show "Following" / "Followers"
- **Mutual relationship:** Show "Connected" badge

**UI Examples:**
```
User A follows User B (one-way):
→ A sees: "Following" button
→ B sees: "Follow Back" button

Both follow each other (mutual):
→ Both see: "Connected" badge
→ Profile: "You're connected with [Name]"
```

**Benefits:**
- Asymmetric following (fan/artist dynamic)
- Mutual connections encouraged (better engagement)
- LinkedIn-like feel when connected
- Simple to understand

**Follow Feed:**
- See posts from users you follow
- Activity feed (follower liked X, commented on Y)
- Discover suggested users based on network

### Wallet & Earnings

**Wallet Features:**
- View balance (available + pending)

- Transaction history

**Earnings Breakdown:**

- Total withdrawn
- Platform spending

**Commission Structure:**
- Basic: 40% platform commission
- Standard: 30% commission
- Premium: 20% commission
- Ultimate: 10% commission
- Ultimate for Friends: 10% commission

**Payout Process:**
- Minimum: $50 balance
- Payout to bank account (Stripe Connect)
- OR payout to PayPal
- Processing time: 2-7 business days (Stripe), 1-3 days (PayPal)
- Automatic payouts (weekly/monthly) OR manual request
- Admin approval required for first payout (fraud prevention)

**All Fees Paid By Sellers:**
- Stripe fees: 2.9% + $0.30
- PayPal fees: ~3.49% + $0.49
- Platform commission: 10-40% (based on tier)
- **Buyers always pay listed price only**

## SPECIALIZED TOOLS

### Collab Matcher

**Purpose:** Match users for collaborations based on complementary skills

**Matching Algorithm:**
- Skills complement (producer + vocalist, guitarist + drummer)
- Genre overlap (both work in hip-hop, both in rock)
- Experience level compatibility
- Location proximity (optional filter - GPS for paid users)
- Availability status

**How It Works:**
1. User clicks "Find Collaborators"
2. System analyzes user's profile (categories, skills, genres)
3. Suggests complementary matches
4. User can filter by genre, location, experience
5. User sends collab request
6. Match can accept/decline

**Collab Request:**
- Includes: Brief message, project idea, timeline
- Receiver can accept, decline, or suggest alternatives

## ADMIN FEATURES

### Content Moderation

**Report System:**
- Report button on all content (posts, products, services, profiles)
- Report categories:
  - Spam
  - Harassment
  - Inappropriate content
  - Copyright violation
  - Scam/fraud
  - Other (with description)

**Moderation Queue:**
- View all reported items
- Review content and user history
- Take action:
  - Dismiss report (false positive)
  - Warn user
  - Remove content
  - Suspend user (temporary)
  - Ban user (permanent)
- Track reporter (false reporting = consequences)

**False Reporting Prevention:**
- Track false report count per user
- 3 false reports = 30-day report ban
- 5 false reports = permanent report ban
- Email notification each time

---

### Custom Field Approval

**User Experience:**
1. User types "Tamburica" (not in skill list)
2. Clicks "+ Request new skill"
3. Saved as pending (**invisible to user**)
4. Admin reviews in panel
5. **Auto-approve if 3+ users request same field**
6. Once approved → appears in checkbox list for everyone
7. User notified: "Your skill 'Tamburica' was approved!"

**Auto-Approval Thresholds:**
- Skills: 3+ users
- Genres: 5+ users
- Categories: Manual only (too critical)

**Admin Review:**
- See all pending requests
- See request count per field
- Approve/reject with reason
- Add to master list (if approved)
- Notify all requesters

**Mobile-First Design:**
- Touch-friendly buttons (44px min)

- Bottom navigation (easy thumb reach)
