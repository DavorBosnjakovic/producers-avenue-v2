# PRODUCERS AVENUE V2 - LAUNCH CHECKLIST

## Overview

Comprehensive checklist for launching Producers Avenue V2 from development to production.

---

## PRE-DEVELOPMENT SETUP

### Development Environment

- [ ] Install Node.js (v18+)
- [ ] Install pnpm/npm/yarn
- [ ] Install Git
- [ ] Set up code editor (VS Code recommended)
- [ ] Install Supabase CLI
- [ ] Install PostgreSQL (for local development)
- [ ] Set up GitHub repository
- [ ] Create project folder structure

### Third-Party Accounts Setup

- [ ] **Supabase:**
  - [ ] Create account
  - [ ] Create new project
  - [ ] Note database URL and keys
  - [ ] Enable Realtime
  - [ ] Set up storage buckets

- [ ] **Stripe:**
  - [ ] Create account
  - [ ] Get API keys (test mode)
  - [ ] Set up Connect (for seller payouts)
  - [ ] Create subscription products
  - [ ] Configure webhooks

- [ ] **PayPal:**
  - [ ] Create Business account
  - [ ] Get API credentials
  - [ ] Enable Payouts API
  - [ ] Configure webhooks

- [ ] **Bunny.net:**
  - [ ] Create account
  - [ ] Create storage zone
  - [ ] Create pull zone (CDN)
  - [ ] Note API keys
  - [ ] Enable replication (2+ regions)

- [ ] **Resend:**
  - [ ] Create account
  - [ ] Verify domain
  - [ ] Get API key
  - [ ] Set up DNS records (SPF, DKIM)

- [ ] **OpenAI:**
  - [ ] Confirm account exists
  - [ ] Get API key
  - [ ] Set usage limits

- [ ] **PostHog or Plausible:**
  - [ ] Create account
  - [ ] Get tracking code/API key

- [ ] **Domain & Hosting:**
  - [ ] Register domain (or use existing)
  - [ ] Set up Hostinger hosting
  - [ ] Configure DNS records
  - [ ] SSL certificate (Let's Encrypt)
